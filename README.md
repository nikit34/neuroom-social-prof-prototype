# Social Proof Homework Prototype

Прототип для проверки точки показа и порога показа для блока:

> Уже сдали из всех получателей

## Что внутри

- Экран "Просмотр задания" с CTA "Сдать ДЗ на проверку"
- Логика показа: `доля сдавших > порог показа`, `получателей >= 8`, опционально "до дедлайна <= 3 дней"
- Переключение между двумя точками показа:
  - Просмотр ДЗ (рекомендуется)
  - Экран отправки фото
- Мини-симуляция по кандидатам порога показа для грубой калибровки

## Запуск

Откройте файл `index.html` в браузере.
